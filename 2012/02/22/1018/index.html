轻描淡写之辞

发表于 2012-02-22 分类于 说时依旧 阅读次数： Valine：
本文字数： 2.3k 阅读时长 ≈ 2 分钟

如果、生命只剩下回忆、我会毫不犹豫的扯出千丝万缕的思絮…

那年、只包齐装、踏上这片土地、那时候的我、也只对这里的残阳与星辰感兴趣、会在乎太阳会在哪个方向落下、会在乎天空中最早闪现的璀璨和最后离去的星光、人家说时间久了、会磨掉一些和生命相关与不相关的事与物、也许偶尔清醒的我、会记下点什么、记下点曾未踏足的空间、只为流年时多点回去的路而已…

给自己置个小院子、开门即是闹市，关门即是深山。后面有条江，弄个小马棚，搞个乌篷船，夜里赏赏月什么的、算是红尘俗世中一片小小的宁静吧…

人总要有一些寄情的地方，如果寄情于一个漂亮妞儿，你想靠近她的时候她不见得想靠近你，但要是寄情于山水、字画、小虫儿这些玩意儿，你随时想亲近它们都可以，没那么麻烦，没那么唧歪。 淡淡的歌、静静的听…

四月的天、写你隆隆的雷声、写你不曾谋面的雨滴、写你清新的空气中、略有点无声的窒息…（每个四月、我都回忆一遍、一直从浅蓝想到深蓝天…） 四月的天、留不住的春眠、挡不住的流炎…

老板、一份回锅饭盖浇肉… 思量、还是思量、哪怕推开窗、透过一丝阳光、但你、却彷徨…彷徨、还是彷徨、哪怕踱步门两旁、穿过一片树行、但你、却思量…故事第一行、你尽写悲凉…

写不尽、后青春期的诗…

念念不忘、此刻应该是农忙、天空弥漫着泥土香、柳青、花绽放、星点白光、我猜想、梨花模样、淡笑、尽思量、念家乡、那是我未完的天堂…

我多想看看那张彩色照片、我多想时间停格青春画面、每个季节我都回忆一遍、一直从浅蓝想到深蓝天…我怀念阳光灿烂睁不开眼、我一直留恋忘返岁月流年、每个夏天我回忆一遍、一直从浅蓝想到深蓝天…

青春如繁花落尽、倘若我足下留情、那踏过的半野、或许、早已花柳成阴…追忆似逝水流年、倘若我留情三笑、那已逝的青春、或许、早已花好月圆…

回不去的、烟雨朦胧、只片伞叶下、遮住的只有、湿透的心…

回忆就像一杯凉茶、片片茶叶都尽泡出它的余香、不管沏壶怎样、杯身怎样、茶味怎样、在我心中、也只不过是一杯、喝不尽的凉茶…

浮伤、多少烟雨楼台、尽留骚客之笔、或凄、或喜…年华、多少车水马龙、尽收看客眼底、或歌、或泣…我在这里、见与不见、只在与她、现与不现…

我以为风吹过的、只是浮沉埃土、却忘了随风而逝的点点思绪、虽然淡、却记忆…

如果、生命只剩下回忆、我会毫不犹豫的扯出千丝万缕的思絮…

我不知道生活对我们做了什么、竟让那么多人义无反顾…

我并不是思想的惰者、滚远点… 我不知道回忆多远、那一去不复返的岁月、纵使我千丝万缕的思念、也只能默数流年…

我不知道该怎么感谢…我亲吻我所有的服饰、我亲吻每一本书本、早餐的包子、总有幸福的味道、我亲吻我自己、把手放在嘴唇上、亲吻生命的味道…感谢…

出门别忘记刮胡子、小朋友很天真、会叫叔叔的…

念家乡、印红、落夕阳，岁月长、枯榕、人模样，西风凉、秋叶、摇飞扬…噫吁兮、念家乡、思断人心肠…

繁花落尽、残留的、只有暗香… 断阳落红、那些许暮色、星星点点、一弯尤物、勾勒出、夜的唯美…

夜阑珊、华灯初上、苍茫大地、尽是漫漫阴霾…辗转间、已是江桥之上、远处渔火、倒行渐灭…长漫漫路、才觉矣、江城已近、学子归来…

喜欢这个季节、喜欢它夕阳的余辉…喜欢这个季节、喜欢它落叶的归属…喜欢这个季节、喜欢它淡淡的清爽…喜欢呆呆看着天空、静静的思念、绵绵的回忆…

我喜欢、这雨夜···朦胧中淡淡的秋意、呼吸着大地的气息、还有那雨滴舞动的旋律··· 夜伴钟磬、那星点思绪、或是眉间印红、暗淡的、似乎萤光闪烁、夜、缠绵… 一叶知秋、了却心头···

天将降大任于是我也、必先抢我饭碗、断我网线、停我手机、空我钱包…其浮乱我所为、此乃、淫贱而遗者、威武而屈者、明朝散发弄秋风、乞以天下、我本将心向明月、奈何明月恋沟渠啊…

何时西风凋碧树、独上顶楼、望尽天涯路、无言已对、回峰路转深宅中、蓦然回首、帘卷西风、人胜黄花瘦…

我还是那么喜欢读别人的故事、却还是读不出自己···

几根稀落的胡须、早已看不出无知年少的轻狂，剩下的只是经纶岁月的沧桑···

莫念秋立夏已去、七月流火伤煞人…

这是一个明恍恍的夏日、无论我如何追寻、那一去不复返的记忆、像伊夏之沫、随风飘逝…纵是我有千丝万缕的心绪、也换不回她眸笑回头的那一瞬…别了红颜、心似忧幽…

狂风暴雨像是在痛苦中的某个天神的哭声、因为他的爱情被大地所拒绝…

我像那夜间之路、正静悄悄地谛听着记忆的足音…

时过境迁、耳边的音乐、还是那么有味道···

有时候真的很累、累到记不起忘了哪些事···

雨后的微风、在阴暗的空中划过…那似曾的颤抖、重新燃起一片篝火…释放了你、燃烧了自己···

微风抚柳、五月将尽…昂首向天、却不见似曾相识燕归来…湖面微波、皎洁明色…低头求索、意旨在欲上青天揽明月…长相思…

夜、微冷，华灯初上、让我往哪里走，去寻找那无尽黑洞的出口···还有谁、会打着灯、一路踏着我的脚印、走着我的路、唱着我的歌、聆听着我的心灵···

五月的天、淡淡的蓝…冒昧侵来的过云雨、总能把人带入深深的思绪中…耳边的旋律、夹杂着雨声…那打在窗上的雨滴、跳动的音符…似曾相识的情景…

窗外的雨滴落的声音，方佛是肖邦的灵源，听不眷的离愁···别是一番滋味在心头···

云舞艳阳照、花丛草木生、生命竟如此美丽、、、 静坐看蝶飞、花见花蝶泪、蝶飞花瓣落花醉、花落花蝶舞蝶裴 。

乍暖还寒时候、最将难息、三身两件棉衣、怎敌它晚来风急、雪过也、正寒时、却是烈日相识…

埋头在自己怀里、感受她的温馨、嘘…别吵醒这美丽编织的梦…

阴天、寒风、却不知冬去春已来；秃树、枯草、却不知五九应对河边柳 …

茶凉了、我再给你续上…
